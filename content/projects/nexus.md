# Nexus Architecture

**Nexus** is a scalable video hosting backend designed to handle high-throughput uploads.

## Tech Stack
* **Runtime:** Node.js & Express
* **Database:** MongoDB (Aggregation Pipelines)
* **Storage:** Cloudinary / Local Storage
